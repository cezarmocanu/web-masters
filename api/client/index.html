🥷 Ce este Web Masters?

Web Masters este un proiect de educație, ce are ca scop principal, creearea de cursuri gratuite de programare, pornind de la concepte simple până la proiecte și tehnologii complexe precum cele ce se întalnesc în cursurile profesionale si universități.

🧑🏼‍🏫 Cine predă cursul?

Numele meu este Cezar, lucrez în domeniul IT de peste 6 ani, iar de mai bine de 2 ani predau în diverse cursuri de certificare IT.

💁‍♂️ Cine poate participa?

  • Adulti, copii, studenti, elevi, toate aceste categorii pot invăța cate ceva din acest curs
  • Persoanelor care doresc să vada ce se afla în spatele programelor și tehnologiei pe care o folosim zi de zi
  • Celor care doresc să își creeze o aplicație web
  • Celor care doresc să urmeze o carieră în domeniul IT

💻 Ce am nevoie pentru acest curs?

  • Disponibilitatea să participi la cel putin una din cele doua sesiuni săptamânale
  • Un laptop sau un PC
  • Multă răbdare, și motivație să exersezi informatiile noi

⭐ Ce include acest curs?

  • Cursuri Începatori

    Ești nou in lumea IT? Află bazele programării, algoritmilor și uneltelor IT urmând cursul de începători.

  • Complet Gratuit

    Scopul Web Masters este cel de a creea materiale GRATUITE de calitate în domeniul IT. Tot ce ai nevoie este un laptop, implicare și curiozitate.

  • Coding Challange

    Dacă ești curios să vezi proiecte mai complexe, precum și curiozități din lumea programării, Coding Challange-ul este ocazia potrivita.

  • Live dar si înregistrat

    În fiecare săptămână vom avea doua sesiuni live, ce vor fi disponibile pe canalul de YouTube Web Masters
